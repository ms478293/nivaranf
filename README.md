## Getting Started

Use pnpm

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Work only on your assigned branch, if no branch is assigned to you, ask Kushal @ IT.

- Never commit directly to the main branch.
- After a new feature is added, run tests on it before pushing it.
- Untested push needs to me marked as untested.
